What is SVG?

Scalable Vector Graphics


Special DOM element with unique Attributes, Tags, & Behaviors Can be used inline and manipulated via SMIL, CSS, & JavaScript

Raster Images (png)

Vector Images (svg)

Key Benefits SVG

  • Smaller file sizes than rendered images
  • Scale to any resolution without distortion
  • SVG compresses incredibly well
  • Crisp on any display

Why SMIL ?

Synchronized Multimedia Integration Language

PROS

  • Provides an inline animation syntax
  • Can animate properties that CSS cannot
  • Decent control over animation triggering/chaining
  • Has better performance than CSS

CONS

  • A bit cumbrous to use
  • Poor IE support

The main elements

  • animate
  • animateMotion
  • animateTransform

Animate

Tag Animate allows you to animate attributes and properties over a period of time.

from="60" to="430" begin="0s" dur="5s"

Codepen

animateMotion

Animates motion along a path you have set.

path="M10,50 q60,50 100,0 q60,-50 100,0"

Codepen

animateTransform

Animates the transform attribute of a shape. The figure moves relative to the coordinates that we asked

from="0 250 150" to="360 250 150"
