The simple game

  • see code

  • view website

Completed:

April 12, 2017

Description:

The RS (Rolling Scopes) Game is an interactive computer game, in which the user explores a series of rooms, passing tests, and dodging obstacles. The set of rooms and obstacles can be extended or replaced to give different game variations.

Product Features:
1. In the simple game, the pupil passes tasks to become an employee of Epam.
2. Player receives award (symbol of Rolling Scopes) for passing the task.
3. Each passing game will have its time to compare it with other.
4. Three lives of hero.
5. Powerful places which give the live.
6. Multiple levels will exist, with varying degrees of difficulty.
7. Defined sequence of tasks

Technology used:

HTML5, CSS3, Javascript, Canvas
